建议先用printf/echo测试下。      
更改光标风格，仅在Termux有效：      
```
"\e[2 q"
"\e[6 q"
"\e[4 q"
```
输出RGB颜色，TTY下无效：
```
"\033[38;2;<R>;<G>;<B>m" #前景
"\033[48;2;<R>;<G>;<B>m" #背景
```
256色：
```
"\033[38;5;<num>m" #前景
"\033[48;5;<num>m" #背景
```
常见：
```
"\n"          #换行
"\b"          #光标退格
"\a"          #响铃(手机是震动)
"\033c"       #清屏
"\033[?25l"   #隐藏光标
"\033[?25h"   #显示光标
"\033[0m"     #关闭属性
"\033[1m"     #高亮
"\033[4m"     #下划线
"\033[8m"     #隐藏
"\033[<num>H" #光标跳到数字行
"\033[<num>G" #光标跳到数字列
```
前景色
```
"\033[30m"  #黑色
"\033[31m"  #红色
"\033[32m"  #绿色
"\033[33m"  #黄色
"\033[34m"  #蓝色
"\033[35m"  #紫色
"\033[36m"  #天蓝
"\033[37m"  #白色
```
背景色
```
"\033[40m"  #黑色
"\033[41m"  #红色
"\033[42m"  #绿色
"\033[43m"  #黄色
"\033[44m"  #蓝色
"\033[45m"  #紫色
"\033[46m"  #天蓝
"\033[47m"  #白色
```
